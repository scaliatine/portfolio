Frost Bank Design File Organization

Problem

Frost Bank, a state-wide bank in Texas, utilizes Figma as their main design software for their User Experience team. And although it allows for greater collaboration the team struggled with identifying the most up-to-date files, how to name files and projects and navigating individual files.

The Solution

After running multiple design workshops, feedback sessions, card sorting exercises and iterations. A new file information architecture was implemented along with new documentation and training for all employees on utilizing the updated structure.
